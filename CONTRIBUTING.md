# 🤝 Contributing to SkillShare

Thank you for your interest in contributing to SkillShare – Community Learning Platform! <br>

We welcome contributions that improve the platform, fix bugs, or add new features.<br>

# 🚀 Getting Started
Fork the repository<br>
Clone your fork:<br>
git clone https://github.com/your-username/skillshare.git<br>
Navigate into the project:<br>
cd skillshare<br>
Install dependencies:<br>

# Backend:

cd server<br>
npm install<br>

# Frontend:

cd client<br>
npm install<br>
Run the project:<br>
npm run dev<br>

# 🌿 Branching Strategy

Create a new branch for every feature or fix:<br>

git checkout -b feature/your-feature-name<br>

Examples:<br>

feature/add-search<br>
fix/login-bug<br>

# 🧑‍💻 Development Guidelines
Follow the existing MVC architecture for backend<br>
Keep code clean and readable<br>
Use meaningful variable and function names<br>
Avoid unnecessary complexity<br>
Handle errors properly<br>

# 📦 Commit Guidelines

Use clear and meaningful commit messages:<br>

Examples:<br>

Added course creation feature<br>
Fixed authentication issue<br>
Improved dashboard UI<br>

# 🔁 Pull Request Process
Push your changes to your branch<br>
Open a Pull Request<br>
Add a clear description:<br>
What you changed<br>
Why you changed it<br>
Add screenshots if UI changes are made<br>

# 🐞 Reporting Issues

Use the GitHub Issues tab to report:<br>

Bugs<br>
Feature requests<br>
Improvements<br>

Please include:<br>

Steps to reproduce the issue<br>
Expected vs actual behavior<br>

# 🔐 Security Guidelines
Do not expose sensitive data (tokens, passwords, API keys)<br>
Follow best practices for authentication and validation<br>

# 💡 Contribution Ideas

You can contribute by:<br>

Adding new features (search, filters, reviews)<br>
Improving UI/UX<br>
Fixing bugs<br>
Optimizing performance<br>
Improving documentation<br>
🙌 Thank You<br>

Your contributions help improve SkillShare for everyone!<br>
